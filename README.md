## Quick Start

### Prerequisites

- Node.js 16+ and npm

### Install & run locally

```bash
git clone https://github.com/xiayulin123/<your-repo-name>.git
cd developerFolio
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).

### Production build

```bash
npm run build
```

Output goes to `build/`. Serve with any static host (Vercel, Netlify, GitHub Pages, etc.).

## Customize Content

Almost everything is configured in **`src/portfolio.js`**:

- `greeting` — name, tagline, resume URL
- `socialMediaLinks` — GitHub, LinkedIn, email
- `skillsSection` — skills bullets & tech icons
- `educationInfo` — school & coursework
- `workExperiences` — co-op roles
- `bigProjects` — projects with `techStack` tags & links
- `achievementSection` — research cards

### Images

Place project/company logos under `src/assets/images/` and reference them with:

```js
image: require("./assets/images/your-image.png")
```

### Demo videos

Put `.mp4` files in `public/videos/` and link from `footerLink`:

```js
{ name: "Demo", url: "/videos/your-demo.mp4" }
```

Works on Vercel and other static hosts when `public/` is deployed.

### Theme colors

Edit `src/_globalColor.scss` (primary purple: `$buttonColor`).

## Deploy to Vercel

1. Push this repo to GitHub.
2. Import the project in [Vercel](https://vercel.com).
3. Framework preset: **Create React App**
4. Build command: `npm run build`
5. Output directory: `build`
6. Deploy.

Ensure `public/videos/` and `src/assets/images/` are committed.

## Optional: GitHub profile section

To enable pinned GitHub repos, create `.env`:

```env
GITHUB_USERNAME=xiayulin123
REACT_APP_GITHUB_TOKEN=your_token
USE_GITHUB_DATA=true
```

Then set `openSource.display: true` in `portfolio.js`.  
`public/profile.json` is gitignored and generated by `fetch.js` on `npm start` / `npm run build`.
